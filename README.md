# AI-Assisted E-commerce Sales & Customer Analytics

## 📌 Project Overview
This project was built to explore how AI tools like Codex and ChatGPT can be used to accelerate real-world data analytics workflows. The goal was to simulate an e-commerce business scenario and apply data analysis techniques to extract meaningful insights.

---

## 🎯 Objective
- Understand how AI-assisted tools can help in data analytics projects
- Analyze sales and customer data to identify trends and patterns
- Track business KPIs such as revenue and conversion rate
- Apply basic customer segmentation and forecasting concepts

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- SQL
- Data Analysis (EDA)
- Basic Forecasting
- AI Tools (Codex, ChatGPT)

---

## 📊 Work Done
- Generated synthetic e-commerce datasets (customers, products, sales)
- Performed data cleaning and exploratory data analysis (EDA)
- Calculated key KPIs:
  - Revenue
  - Conversion Rate
  - Average Order Value (AOV)
- Performed basic customer segmentation (RFM concept)
- Wrote SQL queries for business insights
- Analyzed sales trends and basic forecasting

---

## 🤖 AI Usage (Important)
This project was created as part of experimenting with AI coding tools.

- Used **Codex** to generate initial project structure, datasets, and code
- Used **ChatGPT** to understand logic, improve insights, and refine outputs
- Focus was on learning how to combine **AI + data analytics workflow**
